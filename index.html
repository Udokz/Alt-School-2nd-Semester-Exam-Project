Project Description: Provisioning A Linux Server With A Simple HTML Page

This project involves provisioning a Linux server with a simple HTML webpage.It demonstrates the essential steps involved in setting up a server, intalling the neccessary components required and deploying a static HTML webpage. This documentation provides a detailed overview of the project scope and implentation

Project Objectives

  • To set up a linux Ubuntu server in a cloud-based environment (AWS).
  • To configure the server with basic system updates and essential tools to host a simple HTML webpage.
  • To create and deploy a simple HTML page to the server.

Project Workflow

  • Environment setup with a cloud service provider(AWS).
  • System configuration involving the update and upgrade of the server packages.
  • Configuration and installation of Apache2 web server.
  • Configuration of HTPPS for the Apache2 web server using a free SSL certificate.
  • Creation of a simple HTML page and deployment to the server.
  • Testing of the web server to ensure the HTML file was successfully deployed.

Tools And Technologies

  • Operating System: Ubuntu
  • Web Server: Apache2
  • Deployment Tool: Amazon Web Service(AWS)
  • HTML content
